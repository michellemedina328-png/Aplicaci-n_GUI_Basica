# Aplicación GUI Básica con Tkinter

Este proyecto consiste en el desarrollo de una aplicación de interfaz gráfica de usuario (GUI) utilizando la librería **Tkinter** de Python. La aplicación permite a los usuarios ingresar datos a través de un campo de texto y agregarlos a una tabla visual dentro de la interfaz.

El objetivo principal es aplicar los conceptos básicos del desarrollo de interfaces gráficas, permitiendo la interacción entre el usuario y el sistema mediante componentes visuales como etiquetas, botones, campos de texto y tablas.


# Componentes de la Interfaz

La aplicación utiliza los siguientes elementos de la librería Tkinter:

- **Label (Etiqueta):** para indicar al usuario dónde ingresar información.
- **Entry (Campo de texto):** permite al usuario escribir datos.
- **Button (Botones):**
  - Botón **Agregar** para añadir información a la tabla.
  - Botón **Limpiar** para eliminar datos.
- **Treeview (Tabla):** muestra los datos ingresados por el usuario.


# Funcionalidades

La aplicación implementa las siguientes funcionalidades:

- Ingreso de datos mediante un campo de texto.
- Agregar información a una tabla mediante el botón **Agregar**.
- Mostrar los datos ingresados en la interfaz gráfica.
- Eliminar datos seleccionados o limpiar completamente la tabla mediante el botón **Limpiar**.
- Responder a eventos generados por el usuario, como clics en los botones.


# Tecnologías Utilizadas

- **Python**
- **Tkinter** (Librería estándar para interfaces gráficas en Python)
